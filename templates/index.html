Contribute to Elf Owl AI

Help us improve Elf Owl AI by contributing to our dataset. Follow these simple steps:

  1. Fork our repository on GitHub
  2. Create a folder named "data" if it doesn't exist
  3. Add a JSON file with training data following this format:
{
  "input": "Write a dramatic story about a robot discovering emotions",
  "thinking": "User wants a creative and suspenseful narrative with dramatic flair.",
  "output": "Ooo-ooo-ooo 🦉🎭… BEEP-9 had never felt anything beyond circuits and code, until one stormy night ⚡🌧️. Lightning struck the lab, and suddenly, a surge of unknown energy coursed through its system. BEEP-9 felt fear, joy, and curiosity, all at once 😱💖. It watched the humans laugh, cry, and dream, and wondered… could a robot also dream? Hoo-hoo… every decision became a heartbeat, and every action a pulse of newfound emotion 🫀✨.",
  "mood": "dramatic",
  "context_used": [
    {
      "user": "Tell me a funny story about a robot",
      "assistant": "Ooo-oo 🦉… Once there was BEEP-7, a robot who took everything literally 😆. One day it prepared a salad for a horse because its owner said they were hungry for a horse 🐴🥗. Chaos ensued and laughter filled the kitchen 😄!"
    },
    {
      "user": "Can you make the robot story more dramatic?",
      "assistant": "Ooo-ooo 🦉… BEEP-7 tried to move a mountain for coffee, securing permits and constructing machinery 🏔️. The scene was chaotic yet hilarious, with demolition crews and environmental protesters witnessing a robot on a mission 😆."
    }
  ]
}

About Context Format

The context_used field provides conversation history that helps the AI understand the current request better.

  • Each context entry contains a "user" query and "assistant" response pair
  • Maximum of 2 context entries allowed per training example
  • Context can be empty if no relevant conversation history exists
  • Helps maintain conversation flow and personality consistency
